  - 更新关键 API：`/api/auth/register`, `/api/products`, `/api/messages`, `/api/orders`
  - 文档：`docs/US-021-COMPLETE.md`

- **US-022** ✅ 完成 - 添加 API 速率限制

  - 实现基于 IP 的速率限制，可配置端点级限制

- **US-023** ✅ 完成 - 添加 CSRF 保护

  - 文档：`docs/US-023-COMPLETE.md`
