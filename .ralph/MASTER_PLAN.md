## 一、项目概览

### 1.1 项目定位
山东信息职业技术学院校园数字化服务统一门户 - 整合二手交易、个人课表、商家点评、学校新闻、信息大全五大分站。

### 1.2 技术栈
- **前端**: Next.js 16 + React 19 + TypeScript + Tailwind CSS
- **后端**: Next.js API Routes + Prisma ORM
- **数据库**: SQLite (开发) / PostgreSQL (生产)
- **认证**: NextAuth.js
- **部署**: Vercel

## 六、Ralph 配置优化

### 6.1 推荐配置调整

```bash
# .ralphrc 建议调整

# 增加超时时间 (复杂任务需要更多时间)
CLAUDE_TIMEOUT_MINUTES=25

# 降低电路断路器阈值 (更早检测异常)
CB_NO_PROGRESS_THRESHOLD=2
CB_SAME_ERROR_THRESHOLD=3

# 启用详细日志 (便于调试)
RALPH_VERBOSE=true

# 允许更多工具 (支持完整开发流程)
ALLOWED_TOOLS="Write,Read,Edit,Grep,Glob,Bash(git *),Bash(npm *),Bash(npx *),Bash(node *),Bash(tsx *),Bash(prisma *)"
```

### 6.2 任务执行策略

1. **每次迭代只完成一个用户故事** - 避免任务过大导致超时
2. **强制执行验收标准** - 每个 story 的 acceptance criteria 必须逐项验证
3. **及时更新进度** - 每次迭代后更新 progress.txt 和 prd.json
4. **触发 EXIT_SIGNAL** - 当一个分组完成时，评估是否继续
