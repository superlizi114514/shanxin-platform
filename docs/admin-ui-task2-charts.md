# 后台 UI 改造 - 任务二：数据可视化图表增强

## 任务目标
在仪表板右侧内容区添加丰富的数据可视化图表

## 文件范围
- `src/app/admin/page.tsx` - 仪表板首页（更新图表区）
- `src/app/api/admin/dashboard/charts/route.ts` - 图表数据 API
- `src/components/admin/charts/` - 新建图表组件目录
  - `CategoryPieChart.tsx` - 分类饼图
  - `TrendLineChart.tsx` - 趋势线图
  - `GrowthAreaChart.tsx` - 增长面积图
  - `RadarChart.tsx` - 雷达图

## 实现要求

### 1. 饼状图 - 分类分布
展示内容：
- 商品分类占比（二手商品）
- 商家分类占比
- 用户类型分布

功能：
- 点击扇区查看详情
- _legend_ 点击筛选
- 百分比显示

### 2. 线形图 - 增长趋势
展示内容：
- 近 7 日用户增长
- 近 7 日订单增长
- 近 7 日商品发布

功能：
- 多线条对比
- 时间范围选择（7 天/30 天）
- 鼠标悬停显示数值

### 3. 面积图 - 业务增长
展示内容：
- 累计用户数
- 累计订单数
- 累计交易额

功能：
- 堆叠面积
- 渐变色填充
- 数据导出

### 4. 雷达图 - 平台健康度
展示维度：
- 用户活跃度
- 商品丰富度
- 商家服务质量
- 订单完成率
- 用户满意度
- 内容更新频率

### 5. API 数据格式
```json
{
  "categoryData": [{ "name": "数码", "value": 120 }, ...],
  "trendData": [{ "name": "周一", "用户增长": 50, "订单增长": 30 }, ...],
  "radarData": [{ "subject": "活跃度", "value": 80 }, ...]
}
```

## 验收标准
- [ ] 所有图表使用 recharts 库
- [ ] 响应式设计，适配不同屏幕
- [ ] 数据加载 skeleton 动画
- [ ] 空数据状态友好提示
- [ ] 图表可下载为图片

## 依赖关系
- 无（可独立开发）
