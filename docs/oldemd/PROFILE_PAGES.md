# 个人主页系统说明

## 区分总站个人主页和二手平台个人主页

### 架构说明

```
总站个人主页 (/profile)
├── 基本信息管理
├── 我的发布（来自二手平台）
├── 我的收藏（来自二手平台）
├── 我的课表（来自课表系统）
├── 我的评价（整合二手平台 + 商家点评）
└── 账号安全（总站）

二手平台个人主页 (/products/profile)
├── 二手商品管理
├── 二手订单管理
├── 商品评价
└── 浏览记录
```

### 数据流向

```
┌─────────────────────────────────────────────────────────┐
│                    总站个人主页 /profile                 │
│  (统一门户 - 整合所有分站数据)                               │
└──────────────┬──────────────────────────────────────────┘
               │
       ┌───────┼───────────────┬───────────────┐
       │       │               │               │
       ▼       ▼               ▼               ▼
┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐
│二手平台   │ │课表系统   │ │商家点评   │ │消息通知   │
│/products │ │/schedule │ │/merchants│ │/notifications│
└──────────┘ └──────────┘ └──────────┘ └──────────┘
```

### 实现建议

1. **总站 `/profile` 页面**：作为统一门户，主要是数据展示和导航
2. **二手平台 `/products/profile` 页面**：专注于二手交易的完整功能
3. **数据共享**：总站通过 API 从各分站获取数据，展示汇总信息
4. **操作跳转**：需要修改、删除等操作时，跳转到对应分站页面

### 现有文件

- `/profile` - 当前是二手平台的个人主页，需要改造成总站个人主页
- `/api/profile` - 用户资料 API，需要扩展以支持各分站数据
- `/products/profile` - 二手平台个人主页（新建）
