# 后台 UI 改造 - 任务三：管理页面详情与编辑功能

## 任务目标
为各管理模块添加详情页和编辑功能，实现完整的 CRUD 操作

## 文件范围
- `src/app/admin/users/[id]/page.tsx` - 用户详情页
- `src/app/admin/users/[id]/edit/page.tsx` - 用户编辑页
- `src/app/admin/merchants/[id]/page.tsx` - 商家详情页
- `src/app/admin/merchants/[id]/edit/page.tsx` - 商家编辑页
- `src/app/admin/news/[id]/page.tsx` - 新闻详情页
- `src/app/admin/news/[id]/edit/page.tsx` - 新闻编辑页
- `src/app/admin/guide/[id]/page.tsx` - 指南详情页
- `src/app/admin/guide/[id]/edit/page.tsx` - 指南编辑页
- `src/app/api/admin/*/` - 对应 API 更新

## 实现要求

### 1. 用户管理模块
**详情页展示**：
- 基本信息（头像、昵称、邮箱、手机号）
- 学生信息（学号、校区、系、班级）
- 统计数据（发布商品数、订单数、收藏数）
- 操作记录（登录时间、操作日志）

**编辑功能**：
- 修改基本信息
- 重置密码
- 角色权限调整
- 账号状态（正常/禁用）

### 2. 商家管理模块
**详情页展示**：
- 商家信息（logo、名称、描述、地址）
- 评分数据（平均分、评价数）
- 图片展示（商家环境图）
- 认领状态

**编辑功能**：
- 修改商家信息
- 审核认领申请
- 认证商家
- 上下架管理

### 3. 新闻管理模块
**详情页展示**：
- 新闻内容（标题、正文、封面图）
- 发布信息（作者、发布时间）
- 统计数据（浏览量、评论数）

**编辑功能**：
- Markdown 编辑器
- 图片上传
- 分类选择
- 发布/下架切换

### 4. 信息大全模块
**详情页展示**：
- 文章信息（标题、分类、标签）
- 内容预览
- 浏览统计

**编辑功能**：
- 分类管理
- 标签管理
- 内容编辑
- 发布状态切换

## 设计要求
- 使用 shadcn/ui 组件
- 表单验证使用 zod
- 图片上传支持预览
- 操作成功/失败 toast 提示

## 验收标准
- [ ] 详情页展示完整信息
- [ ] 编辑表单验证完善
- [ ] 支持图片上传预览
- [ ] 操作反馈及时
- [ ] 返回导航正确
