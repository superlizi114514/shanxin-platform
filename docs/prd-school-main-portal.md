# PRD: 山东信息职业技术学院总站门户平台

## 1. 概述

### 1.1 项目背景
建设山东信息职业技术学院统一的门户平台，作为校园数字化服务的总入口。总站采用门户导航式架构，集成二手交易平台、个人课表、商家点评、学校新闻等核心功能模块。

### 1.2 产品定位
- **总站**: 统一导航门户 + 用户认证中心
- **分站**: 各功能模块可独立运行，通过总站统一入口访问

### 1.3 目标用户
- 在校学生（主要）
- 校内/周边商家
- 教职工

---

## 2. 产品目标

### 2.1 核心目标
- 提供统一的校园服务入口
- 实现单点登录，一次登录访问所有分站
- 各分站数据互通，用户体验一致

### 2.2 成功指标
- 用户活跃度 DAU/MAU
- 各分站访问转化率
- 用户满意度评分

### 3.2 用户故事

#### US-001: 总站首页导航
**描述**: 作为山东信息职业技术学院师生，我希望在总站首页看到所有功能入口，以便快速访问需要的服务。

**验收标准**:
- [ ] 首页展示四大核心功能入口卡片
- [ ] 点击入口可跳转到对应分站
- [ ] 显示用户登录状态
- [ ] 未登录用户点击分站跳转到统一登录页

#### US-002: 统一登录认证
**描述**: 作为用户，我希望一次登录后可以访问所有分站，无需重复登录。

**验收标准**:
- [ ] 支持学号/工号登录
- [ ] 登录状态在各分站间共享
- [ ] Token 过期自动刷新或提示重新登录

#### US-003: 二手交易平台
**描述**: 作为学生/商家，我希望发布和浏览二手商品信息。

**验收标准**:
- [ ] 商品发布（标题、描述、价格、分类、图片）
- [ ] 商品浏览（列表/网格视图）
- [ ] 分类筛选（教材、电子、生活等）
- [ ] 关键词搜索
- [ ] 联系卖家（站内消息/联系方式展示）
- [ ] 商品状态管理（在售/已售/下架）

#### US-004: 个人课表 x 地图联动
**描述**: 作为学生，我希望查看课表时能快速定位教室位置，并获得上课提醒。

**验收标准**:
- [ ] 课表展示（周视图/日视图切换）
- [ ] 点击课程显示教室在地图上的位置
- [ ] 教室地图可视化展示
- [ ] 上课前自动提醒（可配置提前时间）
- [ ] 显示距离和预计步行时间

#### US-005: 商家点评
**描述**: 作为学生，我希望查看和发布对学校周边商家的评价。

**验收标准**:
- [ ] 商家列表展示
- [ ] 商家详情页（信息 + 评价 + 评分）
- [ ] 用户发布评价（打分 + 文字 + 图片）
- [ ] 商家排行榜（按评分/热度）
- [ ] 商家分类（餐饮、购物、服务等）
- [ ] 评价点赞/有用数

#### US-006: 学校新闻
**描述**: 作为师生，我希望及时获取学校新闻动态。

**验收标准**:
- [ ] 新闻列表（按时间排序）
- [ ] 新闻分类（学术、活动、通知等）
- [ ] 新闻详情页
- [ ] 热门新闻推荐
- [ ] 新闻搜索功能

## 5. 非功能需求

### 5.1 性能
- 首页加载时间 < 2 秒
- API 响应时间 < 500ms（P95）
- 支持 1000+ 并发用户

### 5.2 安全
- 用户密码加密存储
- API 请求防刷限制
- XSS/SQL 注入防护
- 敏感信息脱敏展示

### 5.3 兼容性
- 支持主流浏览器（Chrome、Edge、Safari）
- 移动端 H5 适配
- 响应式布局

### 5.4 可用性
- 服务可用性 99%+
- 数据定期备份
- 错误日志监控

## 6. 技术架构

### 6.1 技术栈建议
- **前端**: Vue 3 + TypeScript + Vite
- **后端**: Node.js (Express/Hono) / Cloudflare Workers
- **数据库**: PostgreSQL / MySQL
- **存储**: 对象存储（图片/文件）
- **部署**: Vercel / Cloudflare Pages

### 6.2 数据模型（核心）
```
用户表 (users)
- id, 学号，姓名，手机号，头像，学院，专业，班级

商品表 (products)
- id, 卖家 id, 标题，描述，价格，分类，图片，状态

课程表 (courses)
- id, 用户 id, 课程名，教室，时间，周次

商家表 (merchants)
- id, 名称，分类，地址，均分，评价数

评价表 (reviews)
- id, 商家 id, 用户 id, 评分，内容，图片

新闻表 (news)
- id, 标题，内容，分类，发布时，热度
```
